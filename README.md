# 🏆 Tournament Management System

A fully functional desktop GUI application for managing sports and academic tournaments, built with Python and Tkinter.

## 🛠 Tech Stack
- **Language:** Python 3
- **GUI Framework:** Tkinter (ttk)
- **Data Storage:** Text files (participants_data.txt, scoreboard_data.txt)

## ⚙️ Features
- Register individual participants and teams (up to 5 members)
- Manage multiple events: Football, Basketball, Tennis, Literature, Geography
- Automated points allocation (1st = 5pts, 2nd = 4pts, etc.)
- Live scoreboard display
- Unique ID generation per participant/team
- Data persistence between sessions

## 🚀 How to Run
1. Ensure Python 3 is installed.
2. Clone the repository:
3. Run the application:
```bash
   python tournament_system.py
```
No external libraries required — uses Python standard library only.
